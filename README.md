# Installation Commands
```
git clone https://github.com/RaulXH/MetasPloitXH && cd MetasPloitXH

bash MSFinstaller --install
```

# Uninstall Metasploit-framework
```
bash MSFinstaller --uninstall

```
# Update Metasploit
```
bash MSFinstaller --update
```
